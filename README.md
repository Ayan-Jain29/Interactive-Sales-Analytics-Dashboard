# Interactive-Sales-Analytics-Dashboard
## Overview
This project presents an **interactive Sales Analytics Dashboard** built entirely in **Microsoft Excel**.  
The dashboard enables users to select a **specific city** and instantly view detailed performance metrics for that city, including total sales, target achievement, and top-performing sales executives.

---

## Dataset Information
The dataset provides sales performance data for multiple sales executives across various regions.  
It includes the following columns:

- Employee Code  
- Sales Executive Name  
- Region (City)  
- Daily Sales (Day 1 – Day 5)  
- Total Sales  
- Target  
- Target Hit %  
- Away From Target %

Raw data source: `Raw Data.xlsx`  
Dashboard file: `Dashboard_Project.xlsm`

---

## Dashboard Highlights
- Built in **Microsoft Excel** with **VBA Macros** for interactivity.  
- Displays **city-specific analytics** — users can select a city to view its insights.  
- Automatically calculates:
  - Total Sales  
  - Target Achievement %  
  - Performance Gap %  
  - Top and Bottom Performers  
- Interactive charts and KPIs for regional comparison.

## How to Use
1. Download the files:
   - `Raw Data.xlsx`
   - `Dashboard_Project.xlsm`
2. Open `Dashboard_Project.xlsm` in Excel.
3. Enable **Macros** when prompted.
4. Use the **city selection dropdown** to explore sales data for that specific city.
5. View KPIs, charts, and summaries automatically updated based on your selection.
